- Mix questions across levels for a balanced assessment

## 📐 Answer Format (For Contributors)

When adding answers, follow this format:

```markdown
**Q10. What are HTTP status codes?**

**Answer:**  
HTTP status codes are 3-digit codes returned by a server to indicate the result of a request.
- **1xx** — Informational (e.g., 100 Continue)
- **2xx** — Success (e.g., 200 OK, 201 Created)
- **3xx** — Redirection (e.g., 301 Moved Permanently)
- **4xx** — Client Error (e.g., 400 Bad Request, 404 Not Found)
- **5xx** — Server Error (e.g., 500 Internal Server Error)

> 📖 Reference: [HTTP Status Codes — MDN](https://developer.mozilla.org/en-US/docs/Web/HTTP/Status)
```
